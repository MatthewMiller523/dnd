Project:
This calculates averages damages for weapons of d8, d10, 2d6 damage for DnD 2024 (5.5 ed). It also calculates the damages with great weapon master for 2d6 damage, as well as damage rerolls. Rerolls are called adv (advantage), even though these are two different things in the book. d6 damage is actually 2d6 damage, as Matlab variables cannot behind with numbers.

Damages of 1d4, 1d6, and 1d12 are not included but are a simple expansion

Output:
A csv file will all averages is saved to the project directory

Environment:
This runs on Matlab R2025b as of Feb 20, 2026

I wanted to know the hard numbers and needed to test vibe-coding in Matlab.
